# Bicycle Rental Website

This project is a comprehensive Django-based web application with HTML and CSS for the Front-end for bicycle rental, featuring real-time station
search. The platform allows users to register accounts, search and filter bicycles based on location and type, and make online reservations with
flexible rental durations (hourly, daily, weekly). Users can select pick-up and drop-off stations and specify bike types. The system supports
custom payment methods and cost calculation, along with reservation management (reservation and cancellation), and a seamless end-of-ride
process through preferred payment methods.
